## Installation

```bash
# yarn
$ yarn add resumon

# npm
$ npm install resumon
```

## Development

```bash
# install dependencies
$ yarn install

# build for production
$ yarn build

# lint project files
$ yarn lint

# run tests
$ yarn test
```
