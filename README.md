## Installation

```bash
# yarn
$ yarn add ts-result

# npm
$ npm install ts-result
```

## Development

```bash
# install dependencies
$ yarn install

# build for production
$ yarn build

# lint project files
$ yarn lint

# run tests
$ yarn test
```
